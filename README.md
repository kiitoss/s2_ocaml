# Projet OCaml (L1S2)


## Objet du projet

Le projet consiste en une étude de fonctions de tris de listes, et en particulier de leur efficacité
en pratique. Il se compose :
1. d’un certain nombre de connaissances théoriques en programmation et en algorithmique, acquises de manière autonome, pour pouvoir travailler ;
2. d’un ensemble de fonctions qu’il a été demandé de réaliser ;
3. de plusieurs tests permettant de vérifier la correction et l’efficacité des fonctions.


## Explication du projet

Le projet est composé de 5 fonctions de tris de listes différentes indépendantes entre elles. Une étude (sur le temps d'execution notamment) est ensuite réalisée sur ces fonctions de tris afin de les comparer et de faire ressortir la plus efficace.

**L’utilisation de *List.hd* et *List.tl* est globalement interdite.**
